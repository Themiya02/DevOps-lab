# 🚀 Automated CI/CD Pipeline with Self-Healing Monitoring

This project demonstrates an end-to-end DevOps workflow featuring automated deployment and proactive system health management.

## 🛠️ Tech Stack
- **Docker** (Containerization)
- **GitHub Actions** (CI/CD)
- **Python & Flask** (Web Application & Automation)
- **Linux (WSL 2)** (Environment)

## 🌟 Key Features
- **Automated CI/CD:** Code changes are automatically built into Docker images and pushed to Docker Hub.
- **Professional Dashboard:** A dark-themed system monitoring UI.
- **Self-Healing Script:** A Python-based orchestrator that detects container failure and automatically restarts the service.
